macos上的神级输入法squirrel

2018-12-22
mac

Contents

    • 编译安装
    • 参考

squirrel输入法真是极客最想要的输入法，比起所谓的国产输入法，都在忙着收集用户的输入数据，却没有把精力投入到输入法的核心功能。

squirrel目前的优势：

  1. 配置相当丰富，可定制性，可玩性都非常高
  2. 输入速度是非常快
  3. 五笔支持的非常好
  4. 不收集用户数据，这是我最看重的。

编译安装

目前如果要安装squirrel最新版本是要自己编译安装。具体的编译过程如下:

  1. 安装编译工具

    安装Xcode的工具,运行下面的命令

    xcode-select --install

    使用 Homebrew安装相差依赖库:

    # dev tools:
brew install cmake
brew install git

# libraries:
brew install boost

  2. 下载源代码

    git clone --recursive https://github.com/rime/squirrel.git

  3. 编译依赖库

    make deps

  4. 编译squirrel

    make

  5. 安装squirrel

    sudo make install
